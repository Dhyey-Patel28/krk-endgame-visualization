Zoo Visualization Dashboard

A clean visual exploration of the Zoo dataset using a weak baseline preview, a class-by-trait heatmap, and an animal fingerprint wall.

PCA 2D Preview

A supporting baseline only. It is useful mainly because it shows how much overlap remains when the dataset is flattened into a generic 2D projection.

Class by Trait Heatmap

Each row is an animal class. Each column is a trait. The cell value shows the share of animals in that class expressing that trait.

Animal Glyph Wall

A face-plot style view using animal-inspired glyphs. Each animal is drawn with the same base face, while traits change ears, crest, fins, whiskers, teeth, fangs, collar, and other visual parts.

Zoo Similarity Constellation

A similarity galaxy where animals connect to their closest trait-neighbors. Dense neighborhoods reveal natural animal families, while unusual animals act as bridges.

Hover to preview a neighborhood, click a node to lock it, and click empty space to release it.
Hover over an animal to inspect it. Click a node to lock the selection.
